Daruk web Framework

使用 inversifyjs 的 IoC 容器管理依赖，让开发者享受最佳的 OOP 和 IoC 的编程体验。

基于 Koa2

使用 Typescript 开发的轻量级 web 框架,兼容 Koa 生态。

100% 的Typescript

使用 inversifyjs 的 IoC 容器管理依赖，快速上手 Typescript 编程体验。

装饰器函数

大量的装饰器函数让开发者可以写更少的代码来完成更多的事情。

灵活的目录约定

提供给开发者更灵活的操作空间来进行项目约束和编写。

纯 OOP 编程体验

可以更灵活的定制你的 web 项目和编写测试脚本。

配置简单

ts 环境下引入即用，无需脚手架等工具。
